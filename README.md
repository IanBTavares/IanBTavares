# Currículo — Ian Barbosa Tavares

Página HTML de currículo em uma única página, responsiva e pronta para GitHub Pages e export para PDF.

## Estrutura

- `index.html` — Página do currículo
- `styles.css` — Estilos (layout, tipografia, impressão)
- `assets/ian-tavares.png` — Foto de perfil

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex.: `cv-ian-tavares` ou `seu-usuario.github.io`).
2. Envie os arquivos deste projeto para o repositório (branch `main` ou `master`).
3. No repositório: **Settings** → **Pages**.
4. Em **Build and deployment** → **Source**: escolha **Deploy from a branch**.
5. Em **Branch**: selecione a branch (ex.: `main`) e a pasta **/ (root)**.
6. Salve. Em alguns minutos o site estará em:
   - `https://seu-usuario.github.io/cv-ian-tavares/` (se o repositório for `cv-ian-tavares`)
   - `https://seu-usuario.github.io/` (se o repositório for `seu-usuario.github.io`)

## Como gerar PDF

1. Abra `index.html` no navegador.
2. Use **Imprimir** (Ctrl+P) → **Salvar como PDF** ou **Imprimir em arquivo**.
3. Recomendações: desmarque cabeçalho e rodapé; ative “Gráficos de fundo” se quiser manter o layout.

## Editar o conteúdo

Edite diretamente o arquivo `index.html` para alterar texto, experiências e contato. Os estilos podem ser ajustados em `styles.css` (variáveis no topo do arquivo).
